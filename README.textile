h1. LocaleApp

h2. Installation

h3. Rails 3

Add the locale_app gem to your Gemfile :

<pre><code>
gem 'locale_app', :git => 'git@starbug.devmodo.com:gems/olba.git'
</code></pre>

Configure locale_app with the following code in config/initializers/locale_app.rb :

<pre><code>
require 'locale_app/rails'

LocaleApp.configure do |config|
  config.api_key = 'abcdefg1234567890'
end
</code></pre>

h3. Rails 2.3

Define locale_app in config/environment.rb :

<pre><code>
config.gem 'locale_app', :source => 'git@starbug.devmodo.com:gems/olba.git' 
</code></pre>

Install the gem :

<pre><code>
rake gems:install
</code></pre>

Configure locale_app with the following code in config/initializers/locale_app.rb :

<pre><code>
require 'locale_app/rails'

LocaleApp.configure do |config|
  config.api_key = 'abcdefg1234567890'
end
</code></pre>

h2. Automatically pulling translations

In app/controllers/application_controller.rb

<pre><code>
before_filter :handle_translation_updates
</code></pre>
